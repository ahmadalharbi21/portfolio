### 1. **Wellbore and Production Prediction System**
- **Description**: Developed a prediction model for wellbore productivity using environmental and spatial data to assess oil and gas production potential.
- **Highlights**:
  - Built a Gradient Boosting Regressor for production prediction.
  - Visualized spatial data using GIS tools.
  - Focused on environmental monitoring and risk assessment.
- **Tools**: Python, GIS, Gradient Boosting

### 2. **Poetry Recommendation System**
- **Description**: Designed a recommendation system for Arabic poetry using clustering techniques on 100,000 poems scraped from AlDiwan.
- **Highlights**:
  - Used size-constrained clustering for personalized recommendations.
  - Enhanced natural language understanding of Arabic texts.
- **Tools**: Python, Web Scraping, Clustering

### 3. **Chatbot for EXPEC ARC Employees**
- **Description**: Built a chatbot to assist employees with petroleum research and personal development tasks.
- **Highlights**:
  - Integrated OpenAI API for conversational AI.
  - Explored Retrieval-Augmented Generation (RAG) to improve responses.
- **Tools**: Retool, OpenAI API, Flask

### 4. **Stock Price Prediction with LSTM**
- **Description**: Built an LSTM model to predict stock price trends using technical indicators and lagged features for both stock and oil prices.
- **Highlights**:
  - Added features like MACD, RSI, and Bollinger Bands to enhance predictions.
- **Tools**: Python, TensorFlow, Pandas
