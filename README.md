## 🛠 Projects

### 1. Wellbore and Production Prediction System
- **Description**: Developed a prediction model for wellbore productivity using environmental and spatial data to assess oil and gas production potential.  
- **Highlights**:
  - Built a Gradient Boosting Regressor for production prediction.
  - Visualized spatial data using GIS tools.
  - Focused on environmental monitoring and risk assessment.  
- **Tools**: Python, GIS, Gradient Boosting  
- **Note**: This project is not yet published.

### 2. [Poetry Recommendation System](https://github.com/ahmadalharbi21/Poetry-recommendation-system)
- **Description**: Designed a recommendation system for Arabic poetry using clustering techniques on 100,000 poems scraped from AlDiwan.  
- **Highlights**:
  - Used size-constrained clustering for personalized recommendations.
  - Enhanced natural language understanding of Arabic texts.  
- **Tools**: Python, Web Scraping, Clustering  

### 3. [Employee Assistance Chatbot](https://github.com/ahmadalharbi21/Employee-Assistance-Chatbot)
- **Description**: Built a chatbot to assist employees with petroleum research and personal development tasks.  
- **Highlights**:
  - Integrated OpenAI API for conversational AI.
  - Explored Retrieval-Augmented Generation (RAG) to improve responses.  
- **Tools**: Retool, OpenAI API, Flask  

### 4. [Product Price Prediction](https://github.com/ahmadalharbi21/product-price-prediction)
- **Description**: Built an LSTM model to predict product prices based on descriptions and country.  
- **Highlights**:
  - Used embeddings like GloVe for textual features.
  - Incorporated additional features for better accuracy.  
- **Tools**: Python, TensorFlow, Pandas  

### 5. [BLE Device Scanner](https://github.com/ahmadalharbi21/BLE_DEVICE_SCANNER)
- **Description**: A Python-based desktop application for discovering and interacting with nearby Bluetooth Low Energy (BLE) devices.  
- **Highlights**:
  - Scans for nearby BLE devices and displays them in a list.
  - Explores services, characteristics, and properties of BLE devices.  
- **Tools**: Python, Bleak, Customtkinter  

### 6. [Gemini Word Lookup Chrome Extension](https://github.com/ahmadalharbi21/gemini-word-lookup-chrome-extension)
- **Description**: A Chrome extension for quick word definitions and YouTube caption retrieval.  
- **Highlights**:
  - Fetches word definitions and examples using the Gemini API.
  - Retrieves captions from YouTube videos for easy access.  
- **Tools**: Flask, Flask-CORS, Google Generative AI  

### 7. [Motion Detection with OpenCV](https://github.com/ahmadalharbi21/motion-detection)
- **Description**: Detects motion in video streams by comparing consecutive frames and highlighting motion regions.  
- **Highlights**:
  - Processes video frames using OpenCV.
  - Draws bounding boxes around detected motion areas.  
- **Tools**: Python, OpenCV, skvideo  

### 8. [Interactive LED Light System Based on Screen Color](https://github.com/ahmadalharbi21/Interactive-Light-Display)
- **Description**: Synchronizes LED light colors with screen colors in real-time for immersive experiences.  
- **Highlights**:
  - Low-latency lighting using dxcam for screen capture.
  - Bluetooth control for wireless operation.  
- **Tools**: Python, dxcam, Bleak
